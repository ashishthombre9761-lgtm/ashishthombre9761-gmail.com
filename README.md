# Shivleela - Letter Viewer with Animation (Website B)

## 📮 Overview
This is the **Letter Display Website** where users view the apology letters created on Website A. Features a beautiful letter opening animation!

## ✨ Features
✅ Beautiful envelope animation when opening  
✅ Displays all letter content (name, message, photo, audio)  
✅ Permanent access via unique letter ID  
✅ Share button for easy sharing  
✅ Download letter as text file  
✅ Mobile-friendly design  
✅ Back button to open another letter  

## 🎬 How It Works

### Step 1: Scan QR Code
- User scans QR code from Website A
- Gets redirected to this website with letter ID in URL

### Step 2: Letter Animation
- Envelope opens with smooth animation
- User clicks "Click to Open" button
- Letter envelope flap rotates open

### Step 3: View Content
- All letter data is displayed:
  - Sender's name
  - Message text
  - Photo (if included)
  - Audio (if included)
  - Creation date

### Step 4: Share/Download
- Share button to send to others
- Download letter as text file
- Back button to open another letter

## 🚀 Setup Instructions

### Step 1: Firebase Setup
Same as Website A! Follow the Firebase setup in Website A's README.

Once configured, copy the Firebase config to `script.js`

### Step 2: Deploy Website B
Deploy to GitHub Pages:
1. Go to repository Settings → Pages
2. Select "Deploy from a branch"
3. Choose "main" branch
4. Your site will be at: `https://ashishthombre9761-lgtm.github.io/ashishthombre9761-gmail.com/`

### Step 3: URL Structure
Letters are accessed via URL like:
```
https://ashishthombre9761-lgtm.github.io/ashishthombre9761-gmail.com/?id=letter_abc123_1234567890
```

The `id` parameter is generated by Website A
